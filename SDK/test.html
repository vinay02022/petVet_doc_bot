Veterinary Chatbot SDK Test Page

Setup Instructions:
  1. Start MongoDB: mongod
  2. Start Backend: cd BACKEND && npm run dev
  3. Start Frontend: cd FRONTEND && npm run dev
  4. Open this HTML file in your browser

Test 1: Basic Integration

This test loads the chatbot without any configuration.

<script src="chatbot-standalone.js"></script>

The chatbot should appear in the bottom-right corner.
